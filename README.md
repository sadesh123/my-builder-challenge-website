## 📖 Project Overview
This project delivers:
- ✅ A personal "About Me" website hosted on **AWS global network**  
- ✅ **Professional-grade security** with HTTPS  
- ✅ A working **contact form** (secure)  
- ✅ Deployed entirely within **AWS Free Tier**  

---

## 🎨 Features
- Custom theme and layout  
- Interactive sections (fun facts & learning goals)  
- Contact form integrated with AWS services (SES / API Gateway)  
- Globally distributed with **CloudFront** for performance  

--- 

### 📚 Learning Goals
- Strengthen my **AWS builder skills** by deploying end-to-end apps  
- Get hands-on with **security-first architecture**

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Hosting:** Amazon S3 (Static Website Hosting)  
- **Security:** AWS Certificate Manager + CloudFront (HTTPS)  
- **Domain:** Route 53  
- **Contact Form:** AWS API Gateway + Lambda + SES  
- **CI/CD:** GitHub + Amplify
